# 📊 BMI Advanced Calculator

A sleek, dynamic BMI calculator built using React Native and Expo. 

## ✨ Features
- Dynamic color themes based on health categories (Healthy, Overweight, etc.)
- Gender selection cards
- Smooth keyboard management and state clearing

## 🚀 How to Run This Project
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` or `npx expo start` to launch the app
